Для установки HTML-CSS-JS Prettifi переходим в меню Tools - Command Palette и набираем HTML - CSS - JS Prettifi.Пробуем, если вываливается ошибка о не установленном node.js идем в терминал и прописываем: sudo snap install node--classic Затем устанавливаем nvm curl - o - https: //raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash Смотрим версию node--version Смотрим возможные версии nvm list - remote Устанавливаем последнию, например v20 .10 .0 nvm install v20 .10 .0 Дальше идем в папку в Linux путь может быть в формате / home / /.nvm//bin/node(в моем случае /home/vladimiregorov/.nvm/versions/node/v20.10.0/bin/node) Копируем полный путь к node Открываем файл настроек HTML-CSS-JS Prettifi и копируем наш путь вместо выбранного. Сохраняем файл. Пробуем. Все работает.
